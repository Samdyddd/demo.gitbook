book.json配置说明

gitbook 默认自带5个插件：highlight, search, sharing, font-settings, livereload; 要去除自带插件，在插件前加 -

添加插件： 在plugins中添加，在终端gitbook install ./ 即可

{
  gitbook: '指定gitbook版本',
  styles: {
    website: '自定义页面样式'
  },
  structure: {
    "readme": "README.md:指定readme,summary,glossary,languages对应文件名 "

  },
  plugins: [
